# BEAT//BACK 节拍反击

一款受《The Beat Strikes Back》启发的原创网页节拍射击游戏。

## 启动

直接双击 `index.html` 即可游玩。首次进入点击“接入信号”以启用声音。

## 操作

- 点击底部节奏轨：选择对应颜色的炮塔
- 点击右侧节奏卡：每次替换当前炮塔接下来的 4 拍；按 1–4、5–8、9–12、13–16 的顺序从左向右推进，然后重新回到 1–4
- `1`—`4`：快速选择已解锁轨道
- `空格键` 或点击能量区域：消耗 28 能量启动 4 秒过载

击落入侵者获得经验；3 级、6 级会解锁新的声部与炮塔。敌方弹幕触底会损失能量。

黄色炮塔发射高速反射激光：激光带有黄白长尾，碰到顶部或左右边界会反弹，并能连续贯穿最多 3 个敌人。
