# BEAT//BACK 节拍反击

一款受《The Beat Strikes Back》启发的原创网页节拍射击游戏。

## 启动

直接双击 `index.html` 即可游玩。首次进入点击“接入信号”以启用声音。

## 操作

- 点击底部节奏轨：选择对应颜色的炮塔
- 点击右侧节奏卡：每次替换当前炮塔接下来的 4 拍；按 1–4、5–8、9–12、13–16 的顺序从左向右推进，然后重新回到 1–4
- `1`—`4`：快速选择已解锁轨道
- `空格键`：当扫描线与任意轨道的大号实心节拍重合时，所有被扫到大点所对应的炮台都会各自额外发射 1 发；同一个大节拍每轮只能成功触发一次
- `E` 或点击能量区域：消耗 28 能量启动 4 秒过载

扫描线到达音序器最右端后会停留 3 秒，随后回到左侧开始下一轮扫描。停留期间敌人和弹道继续运动，但节拍不会继续触发。

击落入侵者获得经验；3 级、6 级会解锁新的声部与炮塔。敌方弹幕触底会损失能量。

黄色炮塔发射高速反射激光：激光带有黄白长尾，碰到顶部或左右边界会反弹，并能连续贯穿最多 3 个敌人。

敌人死亡后会掉落经验晶体，晶体会自动飞向左下角经验槽。经验达到升级阈值后战斗暂停并进入三选一 Buff 界面；当前 Buff 卡是逻辑占位，选择会被记录并恢复战斗，具体效果将在后续版本中设计。
